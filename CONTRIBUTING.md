# Contributing to Awesome Quantum Computing Programmes

First off, thank you for considering contributing to this repository! It's people like you that make this a great resource for everyone interested in quantum computing education.

## How to add a new programme

We welcome additions of any high-quality, verified quantum computing degree programme, minor, or professional certificate. 

### 1. Check the criteria
- The programme must be officially recognised by an accredited educational institution or a major technology provider (e.g., IBM, Google).
- It must have a clear focus on quantum computing, quantum information science, or quantum engineering.
- The link provided must be active and point directly to the official programme page.

### 2. Submit via Pull Request
1. Fork the repository.
2. Edit the `README.md` file to add the new programme to the appropriate table.
3. If applicable, add the raw data to the corresponding file in the `data/` directory.
4. Ensure you keep the alphabetical or regional sorting order intact when inserting your row.
5. Submit a Pull Request with a brief description of the programme you are adding.

### 3. Submit via Issue
If you aren't comfortable with Pull Requests, you can simply open an Issue using the provided "Add Programme" template. We will review it and add it to the repository for you.

## Updating existing information
If you spot an error (e.g., a broken link, incorrect tuition fee, or a programme that has been discontinued), please open an Issue or submit a Pull Request to correct it.

Thank you for contributing!
